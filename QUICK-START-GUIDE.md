## ğŸ”§ **TROUBLESHOOTING**

### Voice Input Not Working?
- **Check browser**: Use Chrome or Edge
- **Allow microphone**: Grant permission when prompted
- **Check settings**: Ensure mic isn't muted
- **Alternative**: Just type your question!

### Dark Mode Not Persisting?
- **Check browser settings**: Make sure localStorage is enabled
- **Private/Incognito**: Settings don't persist in private browsing
- **Solution**: Use normal browsing mode

### Share Link Not Working?
- **Check URL**: Make sure the full URL was copied
- **Product IDs**: Link should contain `?products=1,2,3`
- **Test**: Try opening in new tab first

### Favorites Disappeared?
- **Clear browsing data**: This deletes localStorage
- **Different device**: Favorites are per-browser, per-device
- **Solution**: Favorite products again (takes 2 seconds!)

### Animations Laggy?
- **Too many products**: Filter to see fewer at once
- **Old device**: Some features need modern hardware
- **Solution**: Still fully functional, just less fancy
