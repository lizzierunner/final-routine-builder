## ğŸ› Troubleshooting

### Chat not responding?
- Check that worker is deployed: \`wrangler deploy\`
- Verify API keys are set in Cloudflare dashboard
- Check browser console for errors (F12)
- Ensure WORKER_URL in script.js matches your deployment

### Achievements not unlocking?
- Check browser's localStorage is enabled
- Open console and run: \`localStorage.getItem('achievements')\`
- Clear localStorage to reset: \`localStorage.clear()\`

### Drag & drop not working?
- Ensure you're clicking the drag handle (â‰¡ icon)
- Check if browser supports drag & drop API
- Try on desktop browser (better support)

### Instagram card not generating?
- Verify HTML2Canvas script is loaded (check Network tab)
- Ensure routine is generated before clicking button
- Check console for canvas errors
- Try on a modern browser (Chrome/Edge/Firefox)

### Natural language search not working?
- Type complete phrases like "affordable moisturizer"
- Try keyword variations from the 70+ supported
- Check if search input has the sparkle hint animation
- Verify products.json loaded correctly

### Products not persisting?
- Check browser's localStorage is enabled
- Clear cache and reload if needed: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
- Check console for storage errors

### Analytics not tracking?
- Ensure localStorage is enabled
- Check if analytics modal shows data
- Try refreshing and performing actions again
- Use Reset button to start fresh if needed
